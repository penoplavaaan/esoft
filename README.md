# .systema
## _Тестовое задание для Esoft_

.systema позволяет пользователям вести контроллировать рабочие процессы.
Есть возможность создавать задачи себе, а также своим подчиненным.
## Стек технологий:
- Laravel
- Bootstrap CSS
- JQuery
- ✨Magic ✨
